# Superstore Sales Analysis Dashboard

## Overview
Interactive Power BI dashboard analyzing Superstore Sales Dataset with 9,801 records across 18 columns.

## Key Insights
- Technology category contributes 36.59% of total sales
- West region generates highest revenue ($0.71M)
- Consumer segment leads with 51% of sales
- Standard Class is most preferred shipping method
- November is peak sales month
- Business shows consistent growth (2015-2018)

## Dashboard Features
- 5 KPI Cards (Total Sales, Orders, Customers, Categories, Regions)
- Sales Trend Analysis (2015-2018)
- Sales by Category Bar Chart
- Sales by Region Bar Chart
- Sales by Segment Donut Chart
- Sales by Ship Mode Bar Chart
- Top 10 Products Analysis
- Interactive Slicers (Year, Region, Category)
- Professional Dark Gold Theme
